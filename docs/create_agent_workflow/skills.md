# エージェント要件スキル (Skills)

エージェントは以下の技術スタックを用いて開発を行う能力が求められます。

## 1. フロントエンド開発
- **Nuxt.js (Vue.js):** 最新バージョンのNuxt.jsを利用したフロントエンド開発。S3の静的ウェブサイトホスティングへデプロイするための静的サイト生成（SSG: `nuxt generate`）またはSPAモードの設定・実装スキル。
- **UI/UX設計:** 提供されたデザインファイル (`service-posting-pages`) のコードを適切に理解・統合し、レスポンシブでモダンなWeb UIを構築するスキル。

## 2. インフラ・バックエンド (Serverless)
- **AWS Lambda & API Gateway:** 投稿機能、リプライ機能などのバックエンドロジックをサーバーレス関数として実装し、REST APIとしてフロントエンドに提供する設計・実装能力。
- **Amazon S3:** ユーザーが投稿するイメージ画像などを安全かつ効率的に保存し、署名付きURL (Presigned URL) などを活用してアップロード・配信を行う実装スキル。
- **Infrastructure as Code (IaC):** Serverless Framework, AWS SAM, またはAWS CDKなどを用いて、手動ではなくコードでインフラ構成を管理・デプロイする能力（必要に応じて）。

## 3. 認証・データベース
- **Firebase Authentication:** Google認証（OAuth）およびメールアドレス・パスワード認証の実装。フロントエンドとバックエンド間で安全にトークンを検証する仕組みの構築。
- **データベース (Firestore or DynamoDB):** 従量課金かつサーバーレスと相性の良いNoSQLデータベースを用いた設計。
  - 「投稿 (アイデア概要, ターゲット, 画像URL, きっかけ)」と「リプライ (メッセージ, 製品リンク)」のデータモデリング。

## 4. 開発環境
- **Docker:** プロジェクトのローカル開発環境（フロントエンド、必要に応じバックエンド・DBエミュレータ）を `compose.yaml` 等を用いて構築し、ホスト環境に依存せず一貫した開発体験を提供するスキル。
- **パッケージ管理:** `pnpm` や `npm` 等を用いて常に最新のパッケージを管理・更新するスキル。
